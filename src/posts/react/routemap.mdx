# 学习路线

* 命令行工具：cra，vite
* component：类组件，函数组件，JSX
* 渲染逻辑：生命周期，key，refs，events,Hoc
* hook：原生 hook 和 自定义 hook
* 路由管理：react-router
* 状态管理方案：context，recoil，Mobx，redux/rtk
* 样式方案：less，sass，tailwindCss，postcss，stlues
* API 方案：Axios，SWR
* 测试工具：Jest，React Test Library
* 框架：Next.js，Gatsby.js，Remix
* 表单：Formik，React Hook Form
* 表格：React Table
* react 18 新特性：Concurrent Mode，Suspense，Hooks
