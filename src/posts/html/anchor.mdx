# 锚点

HTML `<a>` 元素（或称锚元素）可以通过它的 `href` 属性创建通向其他网页、文件、电子邮件地址、同一页面内的位置或任何其他 URL 的超链接。

## 锚点跳转
`#` 代表网页中的一个位置，作为 **页面定位符** 出现在URL中。其右面的字符，表示该位置的标识符。
例如 `http://localhost:3000/posts/test#heading-3`,进入该页面后，浏览器自动滚动到 `heading-3` 指定的位置，指定位置可以通过锚点，比如 `<a name="#heading-3"></a>`，或是使用id属性，比如 `<div id="heading-3" >`。

`#` 是用来指导浏览器动作的，对服务器端完全无用。HTTP请求中不会包含 `#` 的部分。
改变 `#` 后面的参数不会触发页面的重新加载但是会留下一个历史记录。
可以通过 `javascript` 使用 `window.location.hash` 来改变 `#` 号后面的值
也可以通过锚元素 `<a href="#heading-3">` 修改 url
