从 0 到 1 实现一个博客

借鉴

优秀个人独立博客导航 建站于 2006 年，收录了不少个人独立博客。

技术栈

关于博客应用，是最早的互联网应用之一，如今已发展得相当成熟，建站框架也十分多。 如果不关注技术，纯粹想快速建站：

  • WordPress 是一个以 PHP 和 MySQL 为平台的自由开源的博客软件和内容管理系统。 WordPress 具有插件架构和模板系统。 截至 2018 年 4 月，排名前 1000 万的网站中超过 30.6% 使用 WordPress，是世界上最受欢迎的 CMS。
  • Blogger 是由 Google 提供的免费博客平台，使用简单，可以快速创建和发布博客文章。虽然功能相对简单，但对于只需简单博客的用户来说是一个不错的选择。 如果关注技术，想学习知晓整个技术栈：
  • VitePress 是一款基于 vite vue3 的静态站点生成器
  • NextJs 是一个 React 框架，它允许你使用 React 框架建立超强的、有利于 SEO 的、极度面向用户的静态网站和网络应用。
  • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
  • Gatsby is a free and open source framework based on React that helps developers build blazing fast websites and apps
